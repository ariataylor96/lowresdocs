---
title: Sandwich or Dumpling?
---

# Usage

```
!sandwich <username>
```

# Description

Makes a final decision on whether someone is a sandwich or dumpling. If one has already been made, it returns the previous decision.

# Examples

```
!sandwich willowthewhisper
```

# Aliases

 - `!classify`
 - `!dumpling`
 - `!sandwichordumpling`
 - `!issandwich`
 - `!isdumpling`

# Permissions Level

Only the streamer (`3`)

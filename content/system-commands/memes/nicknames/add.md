# Usage

```
!addname <name>
```

# Description

Adds a nickname to the list of things they used to call Nickel back in college.

# Examples

```
!addname Old, busted up crank
```

# Aliases

 - `!newname`
 - `!newnick`
 - `!addnick`
 - `!addcollege`
 - `!newcollegename`
 - `!newcollege`

# Permissions Level

VIP or above (`1`)

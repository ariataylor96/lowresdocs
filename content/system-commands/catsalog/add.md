# Usage

```
!addcatalog <username> <entry>
```

# Description

Adds a new cats-a-log entry for the user provided.

# Examples

```
!addcatalog willowthewhispersr V small demon, primarily composed of coffee.
!addcatalog lesweee (screams of the damned go here)
```

# Aliases

 - `!newcatalog`
 - `!addcatawog`
 - `!newcatawog`

# Permissions Level

Only the streamer (`3`)

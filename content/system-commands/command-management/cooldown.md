---
title: Cooldowns
---

# Usage

```
!setcooldown <commandname> <seconds>
```

# Description

Sets how long has to pass before this command can be called again. By default, this is 2 seconds.

# Examples

```
!setcooldown discord 0
!setcooldown lurk 10
!setcooldown ban 60
```

# Aliases

 - `!settimeout`
 - `!settimer`
 - `!addcooldown`
 - `!addtimer`
 - `!addtimeout`
 - `!modifytimeout`
 - `!modifycooldown`

# Permissions Level

MOD and above (`2`)
